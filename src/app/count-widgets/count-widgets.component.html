Contracts to be Returned more_vert attach_money
Displays all the active/open Rental Contracts which have a model/attachment/part Line(s) delivered but not returned, and whose Rental End Date is less than or equal to Current Date. 5899
Contracts Ready for Delivery more_vert attach_money
Displays all active/open Rental Contracts which have model/attachment/part Line(s) linked to delivery and whose Start Date is less than or equal to the Current Date. 5899
Invoicing Schedules to be processed more_vert attach_money
Checks Rental Invoicing Authorization for all the offices of user and displays un-invoiced Lines which have Scheduled Invoice Date less than or equal to Current Date where Inv. Amount is not equal to zero. 5899
Demo Contracts OverDue more_vert attach_money
Displays active/open Demo Contracts which have a model/attachment/part Line(s) of type Demo which have been delivered but not returned and whose End Date is less than or equal to Current Date. 5899
Demo Contracts Due Today more_vert attach_money
Displays all active/open Demo Contracts which have a model/attachment/part Line(s) of type Demo which have been delivered but not returned and whose End Date is equal to Current Date. 5899
Active Contracts more_vert attach_money
Displays all active/open contracts which have un-invoiced Lines and whose Scheduled Invoice Date is less than or equal to threshold Date. Threshold Date is calculated by adding certain number of days (set in the widget) to Current Date. 5899
Contracts Not Returned more_vert attach_money
Displays all active/open contracts which have any model/attachment/part Line(s) delivered, but have not been returned and the End Date is less than Current Date. 5899
Invoices OverDue and not Paid more_vert attach_money
Displays all invoices of Rental origin, which are present in Open Entries, and have passed the Due Date, where Balance Amount is not equal to zero. 5899
Customer Insurance Expiring more_vert attach_money
Displays the insurance that are expiring in X Days. The number of days is set in the widget configuration 5899
